# 📚 AMO Physics Library Management System

A comprehensive library management system designed specifically for AMO (Atomic, Molecular, and Optical) Physics department. This system allows efficient management of physics books, member registrations, borrowing operations, and PDF ebook downloads with tracking.

## 🌟 Features

### 🎯 Core Features
- **User Authentication**: Separate login for Admins and Members
- **Book Management**: Add, edit, delete, and search physics books
- **Member Management**: Register and manage library members
- **Borrowing System**: Track book borrowing and returns
- **PDF Downloads**: E-book PDF download with tracking
- **Reports & Analytics**: Generate library statistics and reports

### 👨‍💼 Admin Features
- Full control over books and members
- Generate member IDs
- View download statistics
- Export data to Excel
- Librarian tools for manual operations

### 👤 Member Features
- Browse available books
- Borrow physical books
- Download PDF e-books
- View personal borrowing history
- Track personal PDF downloads
